How you can host your website for free on Github

  1. Step 1: First of all you need to create a html file.
  2. Step 2: Go to notepad write down a html code for your website, I think you are smart enough if you don't know how to code.(Use Chat Gpt)😬
  3. Step 3: Save it as index.html (html file extension)>
  4. Step 4: Head on to github.com and create an account.
  5. Step 5: Enter a Repository name, select public then click on "Create a new repository".
  6. Step 6: Click on "upload an existing file".
  7. Step 7: Drag and drop your file or select your index.html file from File Manager.
  8. Step 8: Click on commit changes.
  9. Step 9: Now, go to Setting and select Pages then change the branch None to Main and save it
  10. Step 10: It will take maximum 1-2 minutes to bulid your website.
  11. Step 11: Your site link will appear on Github Pages, the link will appear as "https://Your Github Username.github.io/Your repository name/"
  12. Step 12: That's all you need to do, to own a website.😉
