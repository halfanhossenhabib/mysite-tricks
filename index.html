Host Free Website on GitHub

  1. Create a file: Open Notepad/VS Code and write your HTML code. (Use ChatGPT if you need help 😬).
  2. Save it: Name the file index.html.
  3. GitHub: Go to github.com and create an account.
  4. New Repo: Create a "New Repository", give it a name, and set it to Public.
  5. Upload: Click "Upload an existing file" and drag your index.html in.
  6. Commit: Click the green "Commit changes" button.
  7. Settings: Go to the repo Settings > Pages (on the left sidebar).
  8. Deploy: Change "Branch" from None to Main and click Save.
  9. Wait: It takes 1-2 minutes. Your link will appear at the top!
